# My Sweet Config

*Visualize your dev environment*
<br>
[sweet-config.herokuapp.com][2]

Q3 React Project at Galvanize's Web Development Immersive.

### Usage

```shell
$ yarn install
$ yarn start
```

Note: [API](https://github.com/JBallin/sweet-api) must be running as well.

### Tech Stack
* React.js
* Node.js
* Express.js
* Knex.js
* PostgreSQL
* GitHub API
